# Contextualize Mermaid Syntax

This skill is used for transforming
[Mermaid Diagram Syntax](https://mermaid.js.org/intro/syntax-reference.html) documentation into
token-efficient context for the [diagram skill](../skills/diagram/SKILL.md).

See [SKILL.md](./SKILL.md) for details.

Run `./install.sh` to install this skill to Claude Code.
